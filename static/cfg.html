Configs

Some useful commands when configure your ".cfg" file:

  • bind "key" "command" - Bind a command to a key
  • unbind "key" - Unbind a key-bind command
  • say_team "text" - Print some text in team chat
  • say "text" - Print some text in all chat
  • buy "weapon_name; weapon_name; ..." - Buy weapons
