Server commands

Here is a list of some useful commands when hosting a private server.

Generall commands:
  • ammo_grenade_limit_total "0-5"
  • noclip
  • give "weapon_name"
"sv" commands:
  • sv_cheats "0-1"
  • sv_infinite_ammo "0-1"
  • sv_showimpacts "0-1"
  • sv_grenade_trajectory "0-1"
  • sv_grenade_trajectory_time "x"
  • sv_gravity "x"
"mp" commands:
  • mp_warmup_end
  • mp_roundtime "x"
  • mp_roundtime_defuse "x"
  • mp_restart_game "1"
  • mp_round_restart_delay "0-7"
  • mp_free_armor "1"
  • mp_freezetime "x"
  • mp_limitteams "0-1"
  • mp_autoteambalance "0-1"
  • mp_maxmoney "x"
  • mp_startmoney "x"
  • mp_buytime "x"
  • mp_buy_anywhere "0-1"
  • mp_friendlyfire "0-1"
  • mp_spawnprotectiontime "x"
  • mp_maxrounds "x"
Bot commands:
  • bot_quota "x"
  • bot_add_t
  • bot_add_ct
  • bot_kick
  • bot_stop
